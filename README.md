## flask-bbs

- Python 3 + Flask + Gunicorn + Nginx
- Model部分实现了基于 MongoDB 的 ORM
- Route部分使用 Flask-Blueprint/Flask-Decorator 替换了初始版本的简单实现
- View部分使用 Jinja2 替换了初始版本的模板引擎简单实现

## 已实现功能
- 用户注册/登录，管理员权限，个人管理页面
- 分区/发帖/回复
- 用户私信
- 用户关注
